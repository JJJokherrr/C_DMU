This is a backgammon game, in which a total of six functions a master function composition. 
The six functions are comfort, printing pieces, player moving, computer player running pieces,
finding winners, judging the board is full.

void Init(char broad[ROW][COL]):
This function uses two for nested loops to make the checkerboard empty.

void print(char broad[ROW][COL]):
This function is also initialize the chess board.

void playermove(char broad[ROW][COL]):
This function asks the player to enter the coordinates of the checkerboard above the coordinates

void computermove(char broad[ROW][COL]):
Let the computer run the chess

int Isfull(char broad[ROW][COL]):
Determine if the board is fullS
